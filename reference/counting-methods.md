# Counting Methods

*For any "most/first/only/best" franchise claim, establish the counting method before ranking.*
*This file defines the standard methodologies for the most common claim types.*

---

## Actor Role Counts

When asked how many roles an actor played, the answer depends entirely on which of the following definitions is applied. **Always state which definition you are using.**

### Definition A — Distinct Named Characters
Count only roles where the character has a proper name (first name, surname, or single name) spoken or credited on screen.

*Advantages*: Cleanest, most verifiable.
*Limitations*: Excludes significant work (Vaughn Armstrong's many uncredited aliens; Majel Barrett's computer voice).

### Definition B — All Credited Roles
Count all roles for which the actor received an on-screen or end-card credit, named or unnamed (e.g., "Klingon Warrior," "Alien #2").

*Advantages*: Captures more of the actor's actual contribution.
*Limitations*: Credits are inconsistently assigned across eras; some background roles are credited, equivalent ones are not.

### Definition C — All On-Screen Appearances
Count all roles including uncredited background/extra appearances, verified via episode footage or actor/crew accounts.

*Advantages*: Most comprehensive.
*Limitations*: Very difficult to verify; relies on actor interviews and fan documentation; highly variable confidence.

### Definition D — Voice-Only Roles Counted Separately
For each distinct voice characterization (computer voice, narration, alien voice), count as a separate role.

*Key variable for Majel Barrett*: Under this definition, her computer performances across TOS, TNG, DS9, VOY, ENT, and films are either:
- **One recurring role** (like a recurring character who voices "the computer")
- **Multiple distinct roles** (one per ship, or one per series, or one per appearance)

No official source has resolved this. The researcher should flag it and let the user decide.

### Definition E — Holodeck / Simulation / Alternate Timeline Characters
Count characters that exist only within a holodeck program, simulation, alternate timeline, or dream sequence.

*Key variable for Brent Spiner*: The episode "Emergence" (TNG 7x23) features multiple holodeck characters played by Spiner. Whether these count depends on whether the user considers them "roles" or "effects of a single story."

### Definition F — Canon Only vs. All Media
Restrict to screen canon (TOS through current streaming series, all films) or expand to include animated series (TAS, Lower Decks, Prodigy), Short Treks, and games.

*Note*: TAS was for a period considered non-canon; current official position accepts it as canon. Short Treks status is less clear.

## Canon Boundary Counts

When asked "what counts as canon," the franchise itself has never issued a single definitive statement. Use this working taxonomy:

**Definitively canon:**
- All live-action series (TOS through current streaming)
- All theatrically released films (TOS films through Nemesis; Kelvin trilogy)
- TAS (now generally accepted; previously contested)

**Ambiguous:**
- Short Treks (produced as canon-adjacent; continuity with main series varies)
- Lower Decks, Prodigy (animated; generally accepted as canon by producers)
- In-universe materials (show-produced documents, screen-readable text)

**Not screen canon:**
- Novels, comics, games, RPGs (Memory Beta domain)
- Fan films, fan fiction
- Writer/producer statements about intent that contradict what aired

## "First" Claims

For "first episode/series/film to do X":

1. Establish what X means precisely
2. Check for earlier examples that might qualify under a broader definition of X
3. Note whether the claim is about broadcast order, production order, or stardate order (these occasionally differ)
4. Flag if the "first" claim depends on treating a spin-off as separate from a parent series

Example: "First openly gay character in Star Trek" — requires defining "openly," "gay" (vs. other LGBTQ+ identities), and whether a same-sex-presenting alien species relationship counts.

---

## "Most Appearances" Claims

For character appearance counts:

1. Define "appearance" — credited episode? Scene? Seconds of screentime?
2. Note whether flashback and archival appearances count
3. Note whether voice-only appearances count
4. Clarify series scope
5. Note whether the count is for live-action only or all media

---

## Prime Directive Violations

A perennial fan debate. For any count or ranking:

1. Use the definition of the Prime Directive *as it was understood at the time of the episode* — the directive's stated scope shifted across series
2. Distinguish between violating the letter vs. the spirit
3. Note that Starfleet command sometimes explicitly ordered violations
4. A captain acting under orders is not committing the same infraction as a captain acting unilaterally
